# AI Engineering

A comprehensive guide to AI/ML engineering interviews covering fundamentals, MLOps, and LLM engineering.

## Interview Landscape

AI engineering interviews typically cover:

1. **ML Fundamentals** — Classical ML, deep learning, NLP
2. **System Design for ML** — Training pipelines, serving, feature stores
3. **LLM Engineering** — Prompt engineering, RAG, fine-tuning, agents
4. **Coding** — Implement an algorithm from scratch, data processing
5. **Research Depth** — Understanding recent papers, tradeoffs, evaluation
