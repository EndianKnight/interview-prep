# Model Context Protocol (MCP)

A standard for connecting AI models to data sources and tools.

- **Why:** Avoid N*M integration problem between models and tools.
- **Components:**
    - **MCP Hosts:** Apps like Claude Desktop, IDEs.
    - **MCP Servers:** Provide resources, tools, prompts (e.g., Git, Postgres, Google Drive).
    - **Protocol:** JSON-RPC based standard.
- **Benefits:** Universal connectivity, standardized context.
