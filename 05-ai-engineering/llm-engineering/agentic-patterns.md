# Agentic Patterns

Design patterns for building autonomous agents.

- **ReAct:** Reason + Act loop.
- **Plan-and-Solve:** Generate plan -> Execute.
- **Reflection / Self-Correction:** Critique previous output -> Improve.
- **Multi-Agent Systems:** Manager agent delegates to specialist agents.
- **Memory:** Short-term vs Long-term (Vector DB).
